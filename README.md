# 🌐 Domain IP & Location Tracker (https://domain-intelligence-system-dis.vercel.app/)

This project is a full-stack web application that takes a **domain name** as input, resolves it to its **IP address(es)**, and fetches the **geolocation details** of each IP.

---

## 🚀 Features

- Resolve domain names to IP addresses (DNS lookup)
- Fetch geolocation details (country, city, ISP, etc.)
- Fast backend using FastAPI
- Simple frontend interface for user interaction
- Async API calls for better performance

---

## 🛠️ Tech Stack

### Backend
- FastAPI
- Uvicorn
- dnspython
- httpx
- python-dotenv

### Frontend
- HTML
- CSS
- JavaScript (Fetch API)
